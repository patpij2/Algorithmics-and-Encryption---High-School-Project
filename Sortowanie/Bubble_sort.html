Sortowanie Bąbelkowe (Bubble Sort)

Sortowanie bąbelkowe - inaczej Bubble Sort lub Sinking Sort - polega na porównywaniu dwóch kolejnych elementów i zamianie ich kolejności, jeżeli zaburza ona porządek, w jakim się sortuje tablicę. Sortowanie kończy się, gdy podczas kolejnego przejścia nie dokonano żadnej zmiany.

Sortowanie bąbelkowe ma złożoność czasową n^2, czyli nie jest ona optymalnym algorytmem (ustępuje w wydajności np. Quicksortowi) 

procedure bubbleSort( A : lista elementów do posortowania )

n = liczba_elementów(A)

	do

		for (i = 0; i < n-1; i++) do:

			if A[i] > A[i+1] then

				swap(A[i], A[i+1])

			end if

		end for

		n = n-1

	while n > 1

end procedure
